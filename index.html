Complete Command Reference

üåê Network Commands

netstat - Network Statistics

Display active network connections and listening ports

netstat -tulpn

-t: TCP, -u: UDP, -l: listening, -p: program, -n: numeric

nmap - Network Mapper

Comprehensive port scanning and service detection

nmap -sV -sC -p- -oN scan.txt target.com

-sV: version detection, -sC: default scripts, -p-: all ports, -oN: save output

ss - Socket Statistics

Modern replacement for netstat with better performance

ss -tuln

Faster and more detailed than netstat

ping - Network Connectivity Test

Test network connectivity and measure latency

ping -c 4 target.com

-c: count of packets to send

traceroute - Network Path Analysis

Trace the route packets take to destination

traceroute target.com

Shows all hops between you and target

curl - Transfer Data from URLs

Fetch web pages and test APIs

curl -I https://example.com

-I: fetch headers only, useful for checking responses

üìÅ File Operations

find - Search for Files

Locate files with specific permissions or attributes

find / -perm -4000 -type f 2>/dev/null

Finds SUID files - crucial for privilege escalation

grep - Pattern Matching

Search file contents recursively

grep -ri "password" /var/log/ 2>/dev/null

-r: recursive, -i: case insensitive

chmod - Change File Permissions

Modify file access permissions

chmod 755 script.sh

7=rwx, 5=r-x (owner, group, others)

chown - Change File Owner

Change file ownership

chown user:group file.txt

Change both user and group ownership

tar - Archive Files

Create and extract compressed archives

tar -czvf archive.tar.gz /path/to/directory

-c: create, -z: gzip, -v: verbose, -f: file

üîê Privilege Escalation

sudo -l - List Sudo Privileges

Check what commands current user can run with sudo

sudo -l

First command to run when you have user access

getcap - List Capabilities

Find files with special capabilities set

getcap -r / 2>/dev/null

-r: recursive search from root

find - SUID/SGID Files

Locate files with SUID or SGID bit set

find / -perm -u=s -type f 2>/dev/null

SUID files run with owner's privileges

LinPEAS - Linux Privilege Escalation Script

Automated privilege escalation enumeration

curl -L https://github.com/carlospolop/PEASS-ng/releases/latest/download/linpeas.sh | sh

Comprehensive automated enumeration tool

id - User Identity

Display current user and group information

id

Shows UID, GID, and group memberships

dir: directory mode, -u: URL, -w: wordlist
